
# Project-1: Medical Terms Dictionary 🩺

A simple Python-based dictionary to help users look up medical definitions instantly.


> [!CAUTION]
> **Important Disclaimer:** This dictionary is created using **Gemini AI**. Definitions provided are simplified, one-line summaries and do not cover the full scope of medical terms. They may not always be 100% accurate. Please consult a **medical professional** or a certified medical textbook for clinical use or health concerns.


### Description
This project uses a Python script to create a searchable database of medicinal terms. It takes a word as input from the user and returns the respective definition.

### Inspiration & Methodology
After referencing textbooks and online sources, I found two primary ways to build this:
1. **The Limiter Method:** Using specific symbols as delimiters to parse definitions from a text file.
2. **The Dictionary Method:** Using Python's built-in `dict` structure, assigning the **word** as the `key` and the **definition** as the `value`.

### Features
* 🔍 Quick keyword lookup
* 📖 Accurate medical definitions
* 🐍 Built entirely in Python
